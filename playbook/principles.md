# Principles

Portfolio First

Learning must produce real automation artifacts.

Artifact over Theory

Every session must produce something real.

Telegram-first

Interfaces should use Telegram where possible.

MVP before Complexity

Build simple working systems first.

AI is Optional

Automation comes before AI.

Reliability Matters

Failure paths must be considered.

Human Override

Every system must allow manual intervention.

Commercial Framing

Every artifact should solve a real business problem.
